# mariainsa.github.io

Personal portfolio for Maria Insa, built with React and Vite.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

Create a GitHub repository named `mariainsa.github.io`, upload this project, and enable GitHub Pages with **GitHub Actions** as the source.

The included workflow builds the Vite app and deploys `dist` to GitHub Pages.
